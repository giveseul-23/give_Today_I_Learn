CSS display : blick, inline, inline-block

태그는 block 속성

block 속성은 브라우저 좌측부터 우측까지 영역을 차지(데이터 크기와 관계없이)

block 속성은 크기를 가짐 - width, height 지정

div는 block 속성
일반 텍스트~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

span 태그 inline 속성

inline 속성은 라인내에서 영역을 차지(데이터 크기만큼)

inline 속성은 내용(contents)의 크기로 정해짐

span inline 속성이다. 그래서 한 줄로 표시됨 크기를 지정해도 크기가 달라지지 않는다. 일반 텍스트~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

inline-block 속성

inline-block 속성 라인 내에서 영역을 차지(inline)

inline-block은 전체 영역을 차지하지 않고 한 줄에 표시되지만
가로, 세로 크기를 가진다.
